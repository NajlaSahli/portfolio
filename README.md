## ✏️ كيفيّة التعديل

### تعديل نصّ في ندوة
1. افتحي `content/<اسم_الندوة>.js` (مثل `content/akhlaq.js`)
2. عدّلي النصّ داخل قيمة `html: "..."`
3. احفظي وأعيدي تحميل الصفحة

### استبدال صورة
- ضعي الصورة الجديدة في `assets/figures/`
- غيّري اسم الملف في `src=` داخل `<figure>`

### نقل صورة إلى موضع آخر داخل الندوة
- انسخي كتلة `<figure>...</figure>` كاملة
- الصقيها في الموضع المناسب داخل قيمة `html: "..."`
